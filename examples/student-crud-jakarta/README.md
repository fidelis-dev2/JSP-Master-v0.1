## Run
1. Import `database/schema.sql` into MySQL.
2. Edit DB credentials in `src/main/java/tz/shotto/learn/config/DB.java`.
3. Run `mvn clean package`.
4. Deploy `target/student-crud-jakarta.war` to Apache Tomcat 10.1.
5. Open `http://localhost:8080/student-crud-jakarta`.
