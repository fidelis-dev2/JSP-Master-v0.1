# JSP Jakarta Tomcat Static Learning Website

Professional GitHub Pages ready static website for learning JSP, Servlets, Jakarta, Tomcat, JDBC, MVC, security and deployment.

## Included

- Attractive login page inspired by modern admin dashboards
- Main module screen with app icons, not cards
- Header and collapsible aside/sidebar
- Forgot password popup form using SweetAlert2
- Embedded YouTube player area
- 120 lesson archive files
- Copyable code snippets
- Responsive layout for desktop, tablet and phone
- `.nojekyll` and `404.html` for GitHub Pages

## Deploy to GitHub Pages

Upload the contents of this folder to your repository root, then enable GitHub Pages from Settings > Pages > Deploy from branch > main > root.
